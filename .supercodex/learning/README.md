# Learning

Phase 8 writes recommendation-only learning artifacts here.

- `skills/health.json` stores project-skill health snapshots.
- `patterns/*.json` stores pending pattern candidates.
- `roadmap/*.json` stores per-slice roadmap reassessment reports.
- `process/*.json` stores postmortem-linked process improvement reports.

These artifacts are durable evidence, not automatic edits to the vault or skills.
