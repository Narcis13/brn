# Onboarding

## Local Setup

1. Run `pnpm install`.
2. Run `pnpm test`.
3. Run `pnpm cli doctor`.

## Operator Posture

- Treat `SUPER_CODEX.md` as the normative spec.
- Use `vault/` for durable human-readable memory.
- Use `.supercodex/` for machine state and recovery artifacts.

Store operator and contributor onboarding notes here when the project has enough real workflow to document.
