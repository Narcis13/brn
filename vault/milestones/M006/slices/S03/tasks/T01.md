# T01: Add memory audit reports and automatic audit triggers

## Objective

Generate durable memory audit reports for tasks, slices, and milestones and trigger them at the key Phase 6 boundaries.

## Why Now

Recovery and verification artifacts still drift if the system never checks whether durable summaries match current lower-layer truth.

## Acceptance Criteria

- `audit memory --unit <unit_id>` writes a structured report to `.supercodex/audits/memory/`.
- Task and slice audits detect stale completion and summary mismatches.
- Slice completion and successful recovery resume generate audit reports automatically.
