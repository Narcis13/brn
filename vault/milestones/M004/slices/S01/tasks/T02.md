# T02: Synchronize planning artifacts into the queue

## Objective

Turn validated modern slice and task artifacts into deterministic queue entries without disturbing legacy units.

## Why Now

Modern planning files only become actionable after the deterministic layer can translate them into queue items and dependency edges.

## Acceptance Criteria

- Add or update milestone, slice, and task queue items idempotently from planning artifacts.
- Preserve existing task completion state while blocking tasks behind invalid slice plans.
- Keep new task entries ordered lexically by task ID within a slice.
